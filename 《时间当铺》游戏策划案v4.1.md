# 《时间当铺》游戏策划案 v4.1

> 叙事解谜 / 像素风 / HTML5 Canvas
> 版本：v4.1（对话重写 + 二次来访 + 时间旅人）

---

## 一、项目概述

### 1.1 一句话描述
经营一家收购"时间"的当铺，通过改写客人的过去来改变现在。

### 1.2 类型与风格
- **类型**：叙事解谜 + 经营模拟
- **画面**：像素风（16x20精灵，程序化背景）
- **平台**：HTML5 Canvas，纯浏览器运行，无需安装
- **技术栈**：原生 JavaScript，无框架依赖
- **存档**：localStorage（本地持久化 + 跨周目 MetaSave）

### 1.3 核心体验
玩家扮演一家神秘当铺的老板。客人以"记忆片段"作为抵押，换取改变命运的机会。玩家进入客人的记忆，在关键节点做出选择——每个选择产生"时间涟漪"（Ripple），跨越时空影响其他客人的命运。

---

## 二、核心机制

### 2.1 时间交易
- 客人携带记忆来到当铺，玩家进入记忆场景
- 每段记忆包含一个二选一的关键抉择
- 选择产生时间涟漪，同时获得时间碎片作为报酬
- 时间碎片用于维持当铺运营（延续游戏时限）

### 2.2 蝴蝶效应系统
- **涟漪传播**：每个选择产生的涟漪会影响后续客人的命运
  - 例：陈伯选择学医 → 成为医生 → 救治了林小姐的母亲 → 林小姐不再需要来当铺（被蝴蝶效应阻止）
- **阻断效应**（block_and_replace）：特定涟漪组合可阻止客人出现，替换为蝴蝶效应对话
- **离场对话**（conditionalDeparture）：完成记忆后，基于跨周目涟漪历史触发额外离场对话，揭示角色间隐藏联系
- **共振涟漪**（Resonance）：特定涟漪组合触发共振效果，获得额外时间奖励

### 2.3 当铺经营
收集的时间碎片可用于：
- **延长时限**：打磨沙漏（+50时间），点香炉（+20时间）
- **修复设施**：修复门铃、铜镜、风铃、钟表等，解锁更多客人
- **解锁记忆**：点亮灯笼，解锁老板自身的记忆碎片（主线推进）

### 2.4 分支问候系统（branchingGreeting）
部分客人进入当铺后，玩家可选择不同的开场对话分支：
- **直接分支**（阿蘅、老吴、小周）：播放基础问候后立即显示分支选择
- **延迟分支**（时先生）：先播放完整基础问候（baseGreeting），再进入分支选择
- 分支选择不影响记忆内容，但提供不同的角色背景信息

## 三、叙事设计

### 3.1 叙事结构
- **表面线**：帮助各位客人解决人生遗憾（单元剧形式）
- **暗线**：当铺老板自身的记忆被封锁，每完成一位客人的故事解锁一片记忆碎片
- **主线**：收集全部记忆碎片后发现真相——老板自己曾是时间的典当者
- **真结局**：发现所有客人都与同一条时间线交织，所有选择最终汇聚

### 3.3 角色详细设定

#### 陈伯（陈望山）
- **外貌**：年迈老人，眼中满是沧桑与遗憾
- **记忆**：1962年夏天，18岁的陈望山高中毕业，面前放着入伍通知和医学院录取信
- **选择**：
  - 从军（chen_army）：成为军人，立下战功，失去右臂，一生未娶
  - 学医（chen_doctor）：成为外科医生，1985年救治了一位年轻母亲
- **涟漪影响**：chen_doctor → 阻断林小姐来访（蝴蝶效应）

#### 林小姐（林小玲）
- **外貌**：年轻女子，面容憔悴
- **记忆**：1985年深冬，省城医院，5岁小玲的母亲病重
- **蝴蝶效应**：若陈伯选择学医，林小姐的母亲被救活，她不会来到当铺
- **选择**：
  - 手术（lin_surgery）：手术失败，母亲在手术台上离世
  - 保守治疗（lin_conservative）：母亲多撑了三个月，母女有了最后的时光
- **双问候**：正常问候（greeting_normal）/ 蝴蝶效应问候（greeting_butterfly）

#### 小周（周默）
- **外貌**：年轻人，邮局后勤，鞋上沾着泥，手里攥着一封十年前的信
- **背景**：信封上写着"小玲亲启"，寄件人林远（已故）。12岁时从无法投递的信件堆中偷偷留下这封信，攥了十年
- **记忆**：2010年深秋，县城邮局后院梧桐树下，明天这批无法投递的信就要统一销毁
- **branchingGreeting**：2个分支
  - "信里写了什么？"（branch_letter_content）
  - "为什么等了十年？"（branch_ten_years）
- **选择**：
  - 投入邮筒（zhou_send）：信被送到省城林小玲手中，她哭了一夜，但把父亲照片放在了床头
  - 梧桐树下烧掉（zhou_burn）：灰烬落在树根，来年梧桐开了十年最茂盛的花；周默辞去邮局工作
- **conditionalDeparture**：
  - zhou_send + lin_surgery[meta] → 提到林远的妹妹（林小姐）
  - zhou_burn + linyuan_ghost[hidden] → 提到铜镜中人影

#### 阿蘅（苏蘅）
- **外貌**：白发少女，校服洗得发白，手腕缠着旧红绳。左眼琥珀色，右眼黑瞳
- **背景**：祖父是"守钟人"，维多利亚港钟楼的守护者。她想典当"等待"——让钟继续走或让钟停下的后悔
- **记忆**：1997年香港回归前夜，维多利亚港钟楼上，祖父躺在帆布床上，呼吸微弱
- **branchingGreeting**：3个分支
  - "你祖父是守钟人？"（branch_grandfather）
  - "'等待'怎么典当？"（branch_pawn）
  - "你手腕上的红绳……"（branch_redstring）
- **选择**：
  - 给钟上发条（heng_clock_run）：钟走了，迷路的人找到了回家的路
  - 剪断钟摆绳索（heng_clock_stop）：祖父在沉默中安息
- **conditionalDeparture**：
  - heng_clock_run + chen_army[meta] → 提到祖父说过的独臂恩人
  - heng_clock_stop + chen_doctor[meta] → 提到祖父说过的省城医生

#### 老吴（吴德昌）
- **外貌**：老照相师，指甲缝里有显影液痕迹，怀里抱着一个落灰的铁盒子
- **背景**：1989年，县城照相馆暗房中发现一卷失踪妻子的胶卷——第37号订单，标注"自拍"。是妻子离开前留下的最后影像
- **记忆**：1989年县城照相馆暗房，红色安全灯下，吴德昌面对第37号订单
- **branchingGreeting**：3个分支
  - "冲洗了吗？那卷胶卷"（branch_develop）
  - "你销毁了它？"（branch_afraid）
  - "你的手指……"（branch_fingers）
- **选择**：
  - 冲洗并寄出（wu_develop）：照片显影，是妻子在院子里晒衣服的背影；寄到了妻子老家
  - 烧毁底片（wu_destroy）：底片在药水中溶解；吴德昌说"她在我记忆里比任何照片都清楚"
- **conditionalDeparture**：
  - wu_develop + heng_clock_run[meta] → 提到火车站白发女孩（阿蘅）
  - wu_destroy + heng_clock_stop[meta] → 提到带红绳的女孩

#### 神秘老人（???）
- **外貌**：浑身散发着不属于这个时代的气息
- **记忆**：一段需要解锁全部记忆碎片才能进入的终极记忆
- **涟漪影响**：五条特定涟漪交汇可触发隐藏结局（闭环）

#### 时先生
- **外貌**：灰蓝长袍，银白头发用旧红绳束着。左眼琥珀色，右眼黑瞳
- **身份**：当铺的前任掌柜，120年前用七个人的遗憾创立了时间当铺
- **跨周目三形态**：
  - **首次见面**（shixiansheng_first）：初始形态，讲述当铺的起源
  - **缺席形态**（shixiansheng_gone）：已达成某结局后，铜镜只映出远去的背影
  - **真实形态**（shixiansheng_true）：达成晨光+永恒结局后解锁，进入核心记忆
- **branchingGreeting**：先播放 greeting_true（baseGreeting），再进入3个分支
- **记忆**：当铺创立之日，年轻的时先生面对沙漏，七粒沙代表七段遗憾
- **选择**：
  - 接过沙漏（ripple_accept）：继承当铺，获得旧红绳
  - 打碎沙漏（ripple_reject）：释放一切束缚，获得灰烬残片

#### 时间旅人（隐藏角色）
- **外貌**：军装绿上半身 + 牛仔蓝下半身 + 运动鞋，半白半黑头发
- **身份**：所有未被选择的选择的总和——"第八粒沙，从未存在过的那一粒"
- **触发条件**：metaTrueEndingSeen + metaMaxDays:30 + metaAllCollectibles
- **台词核心**："少年选了从军，我就是那个学医的影子。底片被烧了，我就是那张冲洗出来的照片。"
- **结局**：触发 ending_eighth_sand（隐藏终章·第八粒沙）

## 十四、技术架构

### 14.1 文件结构

```
时间当铺/
├── index.html          # 入口页面
├── js/
│   ├── engine.js       # 引擎层（渲染/输入/粒子/对话/音频）~899行
│   ├── data.js         # 数据层（精灵/客人/故事/背景/成就）~2567行
│   └── game.js         # 逻辑层（场景/系统/状态管理）~2123行
└── 《时间当铺》游戏策划案v4.1.md  # 本文档
```

### 14.2 引擎层（engine.js）
- **PixelRenderer**：Canvas 2D 渲染器，支持像素绘制、精灵渲染、渐变、文字
- **InputManager**：键盘/鼠标输入管理
- **ParticleSystem**：粒子效果（结局、共振等）
- **DialogSystem**：对话框系统，支持逐字显示、分支选择
- **AudioManager**：音效管理（可选）

### 14.3 逻辑层（game.js）
- **MetaSave**：跨周目持久化数据管理
- **AchievementSystem**：成就检测与解锁
- **TitleScene**：标题画面
- **ShopScene**：当铺主场景（经营/对话/隐藏事件检测）
- **MemoryScene**：记忆场景（对话/选择/涟漪/离场对话）
- **ManageScene**：管理界面（升级/记忆碎片/收藏品/成就/状态）
- **AchievementScene**：成就展示界面
- **EndingScene**：结局演出（6种结局+粒子效果）

### 14.4 数据层（data.js）
- **SPRITES**：像素精灵数据（调色板+宽度+字符矩阵）
- **CLIENTS**：客人定义（ID/名称/精灵/解锁条件/报酬）
- **STORIES**：故事/对话数据（问候/记忆/选择/结局对话/分支问候/离场对话）
- **SHOP_ITEMS**：商店升级项目
- **OWNER_MEMORIES**：老板记忆碎片
- **BACKGROUNDS**：程序化背景（Canvas绘制函数）
- **BUTTERFLY_EFFECTS**：蝴蝶效应映射
- **RESONANCE_EFFECTS**：共振涟漪数据
- **HIDDEN_EVENTS**：隐藏事件数据
- **COLLECTIBLES**：收藏品定义
- **ACHIEVEMENTS**：成就定义
- **ACHIEVEMENT_CATEGORIES**：成就分类
